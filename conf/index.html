Congratulations. LCMP is installed successfully!

LCMP Upgrade:

  • Update PHP: yum update -y php-*

  • Update Caddy: yum update -y caddy

  • Update MariaDB: yum update -y MariaDB-*

    • # Change PHP directory's group for Caddy again
    chown root.caddy /var/lib/php/session
    chown root.caddy /var/lib/php/session
    chown root.caddy /var/lib/php/session

    LCMP Uninstall:

  • Uninstall PHP: yum remove -y php-*

  • Uninstall Caddy: yum remove -y caddy

  • Uninstall MariaDB: yum remove -y MariaDB-*

    • Default Location:

  • MariaDB: /var/lib/mysql

  • Web root default location: /data/www/default

    • Process Management:

  • PHP: systemctl [start|stop|status|restart] php-fpm

  • Caddy: systemctl [start|stop|status|restart] caddy

  • MariaDB: systemctl [start|stop|status|restart] mariadb
